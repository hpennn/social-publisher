# 内容发布助手 - Social Publisher

推广内容一键复制发布助手，配合推广工作流使用。

## 功能
- 📋 一键复制推广内容到剪贴板
- 🚀 快速打开各平台发布页面
- 📊 发布状态追踪（已发布/待发布）
- 📱 支持手机浏览器使用

## 支持平台
- 小红书创作者中心
- 知乎创作者中心
- 微信公众号后台
- 抖音创作者中心
- B站创作中心
- 微博

## 使用方式
1. 打开网站
2. 查看今日推广内容
3. 点击「一键复制」复制内容
4. 点击「去发布」选择目标平台
5. 粘贴并发布

## 技术栈
纯静态HTML + CSS + JavaScript，无需构建步骤
